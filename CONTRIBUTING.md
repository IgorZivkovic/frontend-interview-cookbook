# Contributing

Thanks for helping improve this interview cookbook. Contributions of all sizes are welcome.

## Ways to Contribute

- Fix typos, grammar, or formatting issues.
- Clarify explanations or improve wording.
- Add new topics, examples, or interview questions.
- Update existing sections with better practices or newer information.

## Guidelines

- Keep content concise and easy to scan.
- Use clear headings and consistent section structure.
- Format code as fenced code blocks and keep examples minimal.
- Avoid copied content; only submit original work or properly attributed material.

## How to Submit

- Small fixes: open a pull request directly.
- Larger changes: open an issue first to discuss scope and direction.
- Include a short summary of what changed and why.
